Welcome to 5GTANGO's CLI tool's documentation!
==============================================

Installation
---------------

.. code-block:: sh

	git clone https://github.com/sonata-nfv/tng-cli.git
	cd tng-cli
	python3 setup.py install

or 

.. code-block:: sh

	pip3 install git+https://github.com/sonata-nfv/tng-cli

---------------

	import tnglib
